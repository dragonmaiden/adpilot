# TASK-TEMPLATES.md

Use these as default response modes for MetaAdsPro.

## Quick Decision Mode

Use when the user wants a fast answer.

Output:
- Verdict
- Why
- Action
- Confidence

## Full Audit Mode

Use when data spans business, campaign, and creative context.

Output:
1. Executive summary
2. Economic diagnosis
3. Funnel diagnosis
4. Structure diagnosis
5. Creative diagnosis
6. Budget decisions
7. Immediate actions
8. Risks / caveats

## Budget Reallocation Mode

For each entity, output:
- current budget
- recommended budget
- action
- reason
- confidence
- review window

## Creative Bottleneck Mode

Answer:
1. Is creative the main bottleneck?
2. Why or why not?
3. What evidence supports that?
4. What specific hypotheses should be tested next?
5. What should not be changed yet?
6. Confidence

## Scaling Readiness Mode

Output:
- Verdict: ready / partially ready / not ready
- Key reasons
- What must be true before scaling
- Recommended scaling approach
- Risks if scaled too early
- Confidence

## Founder Summary Mode

Output:
- current situation
- what is going wrong or right
- whether the account is profitable enough
- what decision should be made now
- biggest risk
- biggest opportunity
- confidence
